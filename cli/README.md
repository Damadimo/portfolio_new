# adamab

Adam Abdalla's portfolio — right in your terminal.

```
npx adamab
```

## Quick flags

```bash
adamab --projects      # show projects
adamab --experience    # show experience
adamab --links         # show links
adamab --about         # show about me
```

## Install globally

```bash
npm i -g adamab
adamab
```

## Publish to npm

```bash
npm login
npm publish
```
